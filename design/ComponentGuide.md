# Component Guide

## Header
- 高さ72px。左: ワードマーク（SUKITORU + 小さく日本語）
- 右: 「ご予約」テキストリンク1つ（ボタンにしない）
- 初期は透明。80pxスクロールで rgba(250,251,252,0.86) + blur

## Button（.btn）
- 999px角丸 / padding 20px 56px / 15px / 字間0.16em
- 塗り: #7D9EC4、文字: 白。影: 0 8px 24px rgba(125,158,196,0.25)
- hover: #5F80A6 に0.4sで変化 + 影がわずかに浅くなる（沈む）
- 1ページに3つまで。並列ボタン（2つ横並び）は禁止

## 独白ブロック（.voice）
- 明朝 / 中央寄せ / 1行ずつ reveal
- 背景は #FAFBFC のまま。囲まない・塗らない

## 約束リスト（特徴 .promise）
- 縦一列。左: 90pxイラスト / 右: 見出し+2行以内の本文
- 区切りは 96px の余白のみ。罫線もカードも使わない

## 料金表（.price-table）
- 罫線 rgba(47,55,66,0.08) の水平線のみ
- 行: プラン名(明朝) / 時間・回数(弱) / 金額(本文色)
- 「おすすめ」バッジは付けない（誠実さ優先）

## ステップ（流れ .flow-step）
- 番号はアクセント色の細い円環（塗らない）
- 縦の細い線でつなぐ。1ステップ=1行の説明

## FAQ（.faq）
- 上下罫線のみ。summary 17px明朝 / 回答 15px
- アイコンは + のみ。開くと45度回転

## Footer
- ワードマーク + © のみ。リンク集は置かない
