# Animation Guide

参考: Linear / Apple。すべて「呼吸の速度」で。派手さは信頼を削る。

## 基本トランジション（reveal）
- 対象: 各セクションの見出し・段落・図版
- 動き: opacity 0→1 / translateY 24px→0 / blur 8px→0
- 時間: 1.4s / cubic-bezier(0.22, 1, 0.36, 1)
- 発火: IntersectionObserver（threshold 0.2）、1度だけ
- 連続する行は 0.15s ずつ遅延（文字が少し遅れてついてくる）

## 固有の動き
| 場所 | 動き |
|------|------|
| Hero写真 | scale 1.06→1.00 を 8s（ページを開いた瞬間の深呼吸） |
| Hero写真 | スクロールに対し 0.12倍で追従（ごく浅いパララックス） |
| 共感の独白 | 1行ずつ 0.3s 間隔で浮かび上がる |
| 写真セクション | 背景をビューポートより10%大きくし、スクロールでゆっくり移動 |
| ボタン | hover で 1px 沈む影がわずかに深まるのみ。脈動・点滅は禁止 |
| FAQ | 開閉 0.4s ease。+ が × に45度回転 |

## 禁止
- 2秒未満で完結しない無限ループアニメーション（漂う光の玉は廃止）
- バウンド・スプリング・回転入場
- スクロールジャック
- prefers-reduced-motion 時はすべての動きを無効化する
