# Photo Direction

写真が世界観を作る。条件: 自然光 / 生活感 / 余白 / 白・ベージュ・グレー / カメラ目線でない。

## 採用写真（Unsplash・商用利用可）
| 場所 | 写真 | 意図 |
|------|------|------|
| Hero | photo-1519710164239 白い部屋・レースカーテン・観葉植物 | 「安心できる場所」の第一印象。朝の光 |
| SUKITORUとは | photo-1476673160081 淡い朝の海 | ブランドラインの背景。深呼吸の瞬間 |
| 利用者の変化 | photo-1470116945706 重ねられた手 | 唯一の「人の温度」を感じる接写 |
| カウンセラー | photo-1513694203232 グレーの壁・ドライフラワー | 人物写真が入るまでの空気の器 |
| 最後のメッセージ | photo-1507502707541 霧の朝の谷 | 夜明け。「相談は弱さではない」の背景 |

## 加工ルール
- 彩度を上げない。白のベール（rgba白 40〜70%）を重ねて誌面と馴染ませる
- 顔のはっきり写った人物写真は使わない（本人写真が用意できるまで）
- Hero写真は表示時に1.06→1.00へ8秒かけてゆっくり収まる（呼吸）

## 今後の差し替え
- カウンセラー本人の写真（窓辺・自然光・目線外し・上半身）を撮影次第、7章に配置
- 撮影指示: 午前の側光 / 白かベージュの服 / 手元にお茶 / 微笑みすぎない
