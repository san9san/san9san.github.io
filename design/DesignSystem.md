# SUKITORU Design System

## 設計思想
このサイトの目的は情報伝達ではなく「この人なら話してみたい」という感情の醸成。
すべての判断基準は 静けさ / 余白 / 安心感。迷ったら要素を減らす。

## 基本原則
1. 一画面、一感情 — 各セクションは1つのメッセージだけを伝える
2. 余白が語る — 詰めたくなったら余白を増やす
3. 写真が世界観、文字は囁き — 写真7 : 文字3
4. 動きは呼吸の速度 — すべてのアニメーションは1秒以上かけてゆっくり
5. 売らない — CTAは「押させる」のではなく「押したくなる」まで待つ

## レイアウト
- 8px グリッド（すべての余白・サイズは8の倍数）
- コンテンツ最大幅: 1200px（テキストの読み幅は 640px まで）
- セクション上下余白: 180px 以上（モバイル 112px）
- カード角丸: 24px / ボタン: 999px（完全な丸）
- 影: 0 2px 24px rgba(47,55,66,0.05) を上限とする（それ以上濃くしない）

## 禁止事項
- カードの大量陳列 / 情報の詰め込み / 装飾の追加
- ガラスモーフィズムの多用（使用は1箇所まで）
- 派手なアニメーション / 点滅・脈動する要素
- LP的な煽り（「今すぐ」「限定」等）/ 営業的文言

## 品質基準
Apple・Notion・Vercel の製品ページと並べて違和感がないこと。
ただし冷たさは禁止。必ず1セクションに1つ「体温」を残す（手・芽・ハート・肉声）。
