# Color System

## パレット
| 役割 | 値 | 用途 |
|------|-----|------|
| 背景 | #FAFBFC | ページ全体。純白より0.5歩やわらかい |
| 背景・淡 | #F2F5F8 | 静かな区切りに使う淡いブルーグレー |
| 本文 | #2F3742 | 黒ではない。夜明け前の空の色 |
| 本文・弱 | #707B89 | 補足・説明 |
| 本文・微 | #9AA5B1 | ラベル・キャプション |
| アクセント | #7D9EC4 | ボタン・リンク・番号。1画面に1回まで |
| アクセント深 | #5F80A6 | ホバー時のみ |
| 罫線 | rgba(47,55,66,0.08) | 髪の毛の細さで |

## 使用ルール
- アクセントカラーの面積は各画面の3%以下
- 純黒(#000)・純白(#FFF)の文字は使わない
- 写真の上の文字は白 rgba(255,255,255,0.96) + ごく淡い影のみ許可
- イラストの色は本文色系（#7B8FB0）+ 差し色は桜(#F4C0D1)・若葉(#5FA98A)を点で
